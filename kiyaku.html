利用規約

本利用規約（以下「本規約」という。）は、株式会社リベイス（以下、「当社」という。）の提供する「MAKERS BANK」（以下、「当サービス」という。）のご利用にあたり、提供条件並びに当社と当サービスの申込希望者及び利用者である個人及び法人（以下、「ユーザー」という。）当サービス登録によりユーザーへサービス提供を行うフリーランス(以下、「フリーランスユーザー」と言う。)の皆様に遵守していただかなければならない事項が定められております。当サービスの申込希望及び利用に際しては、本規約を必ず全文お読みいただく必要があります。

第１条 適用

本規約は、当社が提供する当サービスの利用に関し、当社とユーザー、フリーランスユーザーとの間 の権利義務関係を定めることを目的とし、当サービスの利用に関わる一切の関係に適用されます。

第2条 定義

本規約において使用する以下の用語は各々以下に定める意味を有するものとします。

  1. 1. 「登録ユーザー」とは、本規約第4条で規定した内容を満たすユーザー、フリーランスユーザーを意味します。
  2. 2. 「本規約」とは、当サービスにおいて規定された本利用規約を意味します。
  3. 3. 「当社」とは、当サービスを運営する株式会社リベイスを意味します。
  4. 4. 「ユーザー」とは、提供条件並びに当社と当サービスの申込希望者及び利用者である個人及び法人を意味します。
  5. 5. 「フリーランスユーザー」とは、当サービス登録によりユーザーへサービス提供を行うフリーランスを意味します。
  6. 6. 「当サービス」とは、当社が運営する 「MAKERS BANK」 を意味します。

第3条 利用規約改定

当社は、ユーザー、フリーランスユーザーの個別合意なしに本規約を改定できるものとする。改定時には当サービス上に掲載し、掲載日より適用されるものとし、改定後掲載日以降に当サービスを利用するユーザー、フリーランスユーザーは本規約の改定に同意したものとみなします。 掲載後、登録ユーザーが本サービスを利用した場合又は当社の定める期間内に登録取消の手続をとらなかった場合には、登録ユーザーは、本規約の変更に同意したものとみなします。

第4条 登録

当サービス登録を希望するフリーランスユーザーは、本規約を遵守することに同意することにより、当社に対し当サービスの登録申請を行うことができます。当社は、当社基準に従いフリーランスユーザーの登録可否を判断し、当社が登録を認める場合にはその旨をフリーランスユーザーへ通知し、この通知以降、登録完了したものとします。

第5条 登録情報の変更

登録後のフリーランスユーザーは、登録情報に変更があった場合は、遅滞なく、当社の定める方法により、当該変更事項について修正を行うこととします。

第6条 禁止事項

ユーザー、フリーランスユーザーが以下の各号のいずれかに該当した場合、当社は事前通知することなく、当サービス利用資格を取り消すことができるものとする。

  1. 1. 当サービスの運営を妨げる行為、その他当サービスに支障をきたす恐れのある行為をした場合
  2. 2. 利用時に、虚偽の内容を申請する行為をした場合
  3. 3. ブログや口コミサイトに、当社の権利、利益、名誉等を損ねる行為、またはそれらの恐れのある行為をした場合
  4. 4. 当サービスの利用において、支払いを不当な理由で免れようとする行為
  5. 5. 当サービスの取引後に、ユーザー、フリーランスユーザーが直接取引を行う行為
  6. 6. 支払期日を遅延し、支払い要求に応えない行為
  7. 7. フリーランスユーザーが連絡返答を遅延した行為
  8. 8. その他、当社が当サービスにおいてユーザー、フリーランスユーザーの行為が不適切と判断した場合

上記に定める他、当社は当社の定める方法でユーザー、フリーランスユーザーへ通知することにより、いつでも当サービス利用資格を取り消すことができるものとする。また、上記5号に記載の直接取引においては事実が発覚した場合、違約金として当該取引金額に相当する金額又は金100万円のいずれか大きい方の金額を当社に支払うものとします。上記6号・7号に該当する行為があった場合、損害賠償金を請求できるものとする。

第7条 サービス

当社は、以下の各号のいずれかに該当した場合、ユーザー、フリーランスユーザーに事前通知せずに当サービスの全部あるいは一部を中断又は停止することができるものとする。

  1. 1. 当サービスに係る設備が事故により停止した場合
  2. 2. 天災地変、第三者の妨害行為等により当サービスの運営が困難になった場合
  3. 3. その他、当社の判断により中断又は停止が必要と判断した場合

本条に基づく措置により、ユーザー、フリーランスユーザーに生じた損害について、当社は一切の責任を負いません。

第8条 本規約の譲渡

当社は当サービスにかかる事業を他社に譲渡した場合には、当該事業譲渡に伴い本規約上の地位、本規約に基づく権利及び義務並びにユーザー、フリーランスユーザーの情報、その他情報を当該事業譲渡の譲受人に譲渡することができるものとし、ユーザー、フリーランスユーザーは、かかる譲渡につき予め同意したものとします。なお、本項に定める事業譲渡には、通常の事業譲渡のみならず、会社分割その他事業が移転するあらゆる場合を含むものとする。

第9条 保証の否認及び免責

  1. 1. 当社は、当サービスの内容の追加、変更、不具合により生じたユーザー、フリーランスユーザーの損害に対し、一切責任を追わないものとします。
  2. 2. ユーザー、フリーランスユーザーが当社から直接又は間接的に当サービス、当サービスを利用するユーザー、フリーランスユーザーその他事項に関する何らかの情報を得た場合であっても、当社はユーザー、フリーランスユーザーに対して本規約を超えて如何なる保証を行うものではない。
  3. 3. 当社は、ユーザー、フリーランスユーザー間においてトラブルがあった場合、本規約の適用がある場合を除き、相互で解決するものとし、当社ではその責任は追わないものとします。
  4. 4. 当社は、当サービスの提供の中断又は停止その他事項においてユーザー、フリーランスユーザーが被った損害につき本規約に該当する場合を除き賠償する責任を一切追わないものとします。
  5. 5. 当社ウェブページに掲載されている広告や当社からのユーザー、フリーランスユーザーに送信する電子メールに掲載されている広告等のリンク先にて生じた損害について、当社は一切の責任を追わないものとします。

第10条 支払い方法

  1. 1. 当サービス利用費用の支払い期日、支払い指定先は、別途当社が定めユーザー、フリーランスユーザーに通知するところに従うものとします。
  2. 2. 当社が事前に承諾した場合を除き、ユーザー、フリーランスユーザー間で当サービス利用費用の支払いを行うことは禁止されるものとします。
  3. 3. ユーザーが当社へ支払いを行う際の振込手数料はユーザーが負担するものとします。また、当社がフリーランスユーザーへ支払いを行う際の振込手数料は当社が負担するものとします。
  4. 4. フリーランスユーザーが納品月から3ヶ月間、請求書発行及び請求業務に応じなかった場合、支払いは無効とする。

第11条 分離可能性

本規約のいずれかの条項又はその一部が、消費者契約法その他の法令等により向こう又は執行不能と判断された場合であっても、本規約の残りの規定及び一部が無効又は執行不能と判断された規定の残りの部分は、継続して完全に効力を有するものとします。

第12条 準拠法及び管轄裁判所

本規約の準拠法は、日本法とします。本規約に起因し又は関連する一切の紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

第13条 協議解決

当社及び、ユーザー、フリーランスユーザーは本規約に定めのない事項又は本規約の解釈に相違が生じた場合には、協議の上速やかに解決を図るものとします。
